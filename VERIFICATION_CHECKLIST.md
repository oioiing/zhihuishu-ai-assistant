## 📋 测试步骤

### 步骤1: 安装扩展
```bash
git clone https://github.com/oioiing/zhihuishu-ai-assistant.git
cd zhihuishu-ai-assistant
npm install
npm run build
# 在Chrome中加载扩展
```

### 步骤2: 访问作业页面
访问: https://hike-teaching-center.polymas.com/pre-space-hike/homeworkCorrect/3676968/862821523/0

### 步骤3: 启动批改
1. 点击页面右侧悬浮球
2. 选择"✅ 作业批改"
3. 等待系统识别题目
4. 点击"🚀 开始批改所有题目"

### 步骤4: 验证结果
1. 检查每题是否有反馈卡片
2. 验证得分是否合理
3. 查看统计数据是否准确
4. 尝试导出报告

### 步骤5: 配置测试
1. 修改批改规则
2. 重新批改验证效果
3. 测试AI服务（如已配置）
