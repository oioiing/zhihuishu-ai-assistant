# 智慧树 AI 助教 - 快速使用指南

## 🚀 5分钟快速上手

### 第一步：安装插件

1. 打开Chrome浏览器
2. 访问 `chrome://extensions/`
3. 开启"开发者模式"（右上角开关）
4. 点击"加载已解压的扩展程序"
5. 选择 `zhihuishu-ai-assistant` 文件夹
6. 安装完成！✅

### 第二步：配置API密钥

1. 访问 https://platform.deepseek.com
2. 注册并获取API密钥（格式：sk-xxxxxxxx...）
3. 打开 `background.js` 文件
4. 找到 `const API_KEY = "sk-..."`
5. 替换为你的真实API密钥
6. 保存文件并重新加载扩展

### 第三步：开始使用

1. 访问任意网页（建议先访问智慧树网站）
2. 右侧会出现📚悬浮球
3. 点击悬浮球展开功能菜单
4. 选择你需要的功能

---

## 📖 功能使用教程

### 🎯 智能识别作业（推荐）

**适用场景：** 快速批改网页上的作业内容

**操作步骤：**
1. 打开包含作业的网页
2. 点击悬浮球 → 选择"🎯 智能识别作业"
3. 等待系统自动识别（1-2秒）
4. 从识别结果中选择要批改的区域
5. 系统自动截图 → OCR识别 → AI批改
6. 查看批改结果

**提示：**
- 如果识别不准确，可以使用"📸 手动截图"
- 识别前确保页面完全加载
- 支持识别多个作业区域，选择最合适的

---

### ⚙️ 自定义批改规则

**适用场景：** 根据不同作业类型设置批改标准

**操作步骤：**
1. 在智能识别页面点击"⚙️ 批改设置"
2. 选择批改重点（可多选）
3. 选择批改严格程度
4. 输入自定义批改指令（可选）
5. 点击"💾 保存设置"

**批改重点说明：**
- **语法**：检查语法错误、时态、语态等
- **拼写**：检查单词拼写错误
- **标点**：检查标点符号使用
- **词汇**：评估词汇丰富度和准确性
- **句式**：分析句子结构和多样性
- **逻辑**：检查论证逻辑和连贯性
- **格式**：检查格式规范
- **内容**：评估内容质量和相关性

**批改严格程度：**
- **宽松**：鼓励为主，适合低年级或初学者
- **适中**：平衡指导，适合大多数情况（默认）
- **严格**：细致批改，适合高年级或重要作业

**自定义指令示例：**
```
重点关注动词时态的使用，忽略小的拼写错误
```
```
多鼓励学生，指出文章的亮点和进步
```
```
严格检查语法错误，提供详细的修改建议
```

---

### 📸 手动截图批改

**适用场景：** 智能识别失败或需要精确选择区域

**操作步骤：**
1. 点击悬浮球 → 选择"📸 截图批改"
2. 屏幕变暗，出现十字光标
3. 按住鼠标左键拖拽选择区域
4. 松开鼠标完成选择
5. 系统自动截图 → OCR识别 → AI批改
6. 查看批改结果

**提示：**
- 选择区域要包含完整的文字内容
- 避免选择过小的区域（最小50x50像素）
- 按ESC键可以取消截图

---

### 💬 AI对话

**适用场景：** 教学答疑、学习指导

**操作步骤：**
1. 点击悬浮球 → 选择"💬 AI对话"
2. 在输入框输入问题
3. 点击"发送"或按回车键
4. 等待AI回复
5. 继续对话

**示例问题：**
- "如何提高英语写作水平？"
- "现在完成时和过去完成时的区别是什么？"
- "请帮我分析这个句子的语法结构"

---

### 📊 页面分析

**适用场景：** 快速了解页面主要内容

**操作步骤：**
1. 点击悬浮球 → 选择"📊 页面分析"
2. 等待AI分析（3-5秒）
3. 查看关键词和页面摘要

**分析内容：**
- 核心关键词（带权重）
- 页面摘要
- 主要内容概述

---

### 📄 页面摘要

**适用场景：** 生成结构化的课程分析报告

**操作步骤：**
1. 点击悬浮球 → 选择"📄 页面摘要"
2. 等待AI生成摘要（5-10秒）
3. 查看Markdown格式的分析报告

**报告内容：**
- 课程档案（教师、学分等）
- 核心内容总结
- 知识图谱（核心模块）
- 学习攻略（难点预警）

## 🎯 最佳实践

- 首次使用前先设置批改规则
- 使用智能识别功能提高效率

## 🔧 常见问题解决

### 问题1：智能识别找不到作业区域
**解决方案：**
1. 确保页面完全加载
2. 检查页面是否包含作业内容
3. 使用"📸 手动截图"功能
4. 刷新页面后重试

### 问题2：OCR识别不准确
**解决方案：**
1. 确保截图区域清晰
2. 文字大小适中（不要太小）
3. 背景与文字有足够对比度
4. 避免选择模糊或倾斜的文字

### 问题3：AI批改失败
**解决方案：**
1. 检查API密钥是否正确
2. 确认账户余额充足
3. 检查网络连接状态
4. 查看浏览器控制台错误信息

### 问题4：批改规则不生效
**解决方案：**
1. 确保点击了"💾 保存设置"
2. 刷新页面后重新批改
3. 检查浏览器存储权限
4. 尝试"🔄 恢复默认"后重新设置

### 问题5：悬浮球不显示
**解决方案：**
1. 刷新页面
2. 检查扩展是否启用
3. 查看浏览器控制台错误
4. 重新加载扩展
