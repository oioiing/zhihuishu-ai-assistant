# 更新日志 (Changelog)

## [1.0.1] - 2026-02-06

### 🔒 安全修复
- **移除硬编码的 API 密钥**：所有 API 密钥现在通过 `chrome.storage.sync` 安全存储
- **添加 .gitignore**：防止敏感配置文件被意外提交到代码仓库
- **创建配置模板**：添加 `config.example.js` 作为配置参考

### ✨ 新增功能
- **设置界面**：在 popup 中添加完整的 API 密钥配置界面
- **调试模式**：支持在设置中开启/关闭调试日志
- **配置重载**：修改设置后自动重新加载配置，无需重启扩展

### 🐛 Bug 修复
- **OCR API 密钥**：修复使用占位符 'helloworld' 导致 OCR 功能不可用的问题
- **DeepSeek API 调用**：所有 API 调用现在使用存储的密钥而非硬编码
- **URL 匹配**：添加对 .cn 域名的支持（*.zhihuishu.cn/*）
- **错误提示**：改进 API 密钥未配置时的错误提示信息

### 📚 文档更新
- **README.md**：更新安装和配置说明，添加详细的故障排查指南
- **INSTALLATION.md**：创建完整的安装和使用指南
- **安全说明**：添加 API 密钥安全使用说明

### 🔧 代码改进
- **配置管理**：实现统一的配置获取函数 `getApiKeys()`
- **调试日志**：添加 `debugLog()` 函数，支持条件输出日志
- **API 验证**：改进 API 密钥格式验证和错误处理
- **代码清理**：移除所有硬编码的敏感信息

### 🧪 测试工具
- **validate.js**：添加自动验证脚本，检查扩展结构和安全性

### 🎯 改进点
- 提高了安全性：API 密钥不再暴露在代码中
- 改善了用户体验：提供图形化的配置界面
- 增强了可维护性：统一的配置管理和错误处理
- 优化了调试体验：可控的调试日志输出
