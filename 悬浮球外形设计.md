# Teacher Help 插件 - 悬浮列表展开版外形设计文档

## 🎨 设计概览

Teacher Help 插件采用**悬浮列表展开**设计，以简洁的圆形触发按钮为入口，点击后优雅展开为垂直功能列表。设计理念追求**极简交互**，通过流畅的展开动画和圆润的视觉元素，提供直观且美观的功能访问体验。

## 📐 核心尺寸规格

### 主触发球体
- **直径**: 48px × 48px
- **形状**: 完美圆形 (border-radius: 50%)
- **位置**: 固定在屏幕右侧，距离边缘 20px
- **层级**: z-index: 2147483647 (最高层级)

### 展开列表容器
- **宽度**: 200px
- **高度**: 自适应内容 (约 240px)
- **圆角**: 16px (四角圆润)
- **位置**: 相对于触发球体左侧展开
- **间距**: 距离球体 12px

## 🎭 交互状态设计

### 主球体状态

#### 默认状态 (Idle)
- **中心图标**: 📚 (代表教育/学习)
- **大小**: 48px × 48px
- **透明度**: 完全可见
- **状态**: 静止，等待点击

#### 悬停状态 (Hover)
- **缩放**: `transform: scale(1.05)` (放大5%)
- **阴影**: 加深阴影效果
- **过渡**: `transition: 0.2s ease`
- **提示**: 显示"点击展开功能"

#### 激活状态 (Active/Expanded)
- **图标变化**: 📚 → ✕ (关闭图标)
- **旋转**: `transform: rotate(45deg)` (45度旋转)
- **颜色**: 背景色稍微加深

### 展开列表状态

#### 展开动画 (Expand)
```css
@keyframes expandList {
  0% {
    opacity: 0;
    transform: translateX(20px) scale(0.8);
  }
  100% {
    opacity: 1;
    transform: translateX(0) scale(1);
  }
}
```
- **持续时间**: 0.3s
- **缓动**: `cubic-bezier(0.16, 1, 0.3, 1)`
- **效果**: 从右向左滑入 + 缩放

#### 收起动画 (Collapse)
```css
@keyframes collapseList {
  0% {
    opacity: 1;
    transform: translateX(0) scale(1);
  }
  100% {
    opacity: 0;
    transform: translateX(20px) scale(0.8);
  }
}
```

## 🎬 完整动画序列

### 展开序列
1. **点击球体** → 球体图标旋转45度变成关闭图标
2. **列表出现** → 从右向左滑入，同时缩放从0.8到1.0
3. **列表项依次显示** → 每个项目延迟50ms出现 (stagger effect)

### 收起序列
1. **点击球体或外部** → 列表项依次消失
2. **列表消失** → 向右滑出，同时缩放到0.8
3. **球体恢复** → 图标旋转回原位，变回📚

### 功能激活序列
1. **点击列表项** → 项目高亮，列表收起
2. **球体状态变化** → 显示对应功能色的加载环
3. **功能面板出现** → 从中心向外展开

## 🖱️ 交互逻辑

### 主要交互流程
1. **默认状态**: 显示圆形球体，中心📚图标
2. **点击球体**: 
   - 球体图标旋转45度变成✕
   - 左侧展开功能列表
   - 列表项依次出现
3. **悬停列表项**: 对应项目高亮显示
4. **点击列表项**: 
   - 触发对应功能
   - 列表收起
   - 显示功能面板
5. **点击球体或外部**: 收起列表，恢复默认状态

### 鼠标事件映射
- **球体点击**: 切换展开/收起状态
- **列表项悬停**: 高亮效果 + 轻微缩放
- **列表项点击**: 功能激活 + 列表收起
- **外部点击**: 自动收起列表
- **球体拖拽**: 改变位置（长按触发）

### 键盘支持
- **Tab**: 在功能项间切换焦点
- **Enter/Space**: 激活当前焦点项
- **Escape**: 收起展开的列表

## 🔧 技术实现要点

### 关键CSS技术
- **Backdrop-filter**: 毛玻璃背景效果
- **Transform**: 缩放、旋转和位移动画
- **Transition**: 平滑过渡效果
- **Animation**: 复杂的展开收起动画
- **Border-radius**: 圆润的边角设计

### JavaScript交互控制
```javascript
// 展开/收起状态管理
let isExpanded = false;

// 切换展开状态
function toggleExpand() {
  isExpanded = !isExpanded;
  if (isExpanded) {
    showFunctionList();
    rotateBallIcon(45); // 旋转成关闭图标
  } else {
    hideFunctionList();
    rotateBallIcon(0); // 恢复原始图标
  }
}

// 显示功能列表
function showFunctionList() {
  const list = createFunctionList();
  document.body.appendChild(list);
  // 添加stagger动画延迟
  list.querySelectorAll('.zh-function-item').forEach((item, index) => {
    item.style.animationDelay = `${index * 50}ms`;
  });
}
```

### 性能优化策略
- **GPU加速**: 使用transform而非left/top
- **事件委托**: 高效的事件处理
- **动画优化**: 使用will-change属性
- **内存管理**: 及时清理DOM元素
