# 功能说明（当前版本）

## 1. 项目定位

本插件是面向知慧树教师场景的 Chrome 扩展，核心目标是提升作业批改效率，覆盖“作业分析 → 批改执行 → 结果查看与诊断”完整流程。

## 2. 当前功能清单

### 2.1 批改与分析能力

- AI 作业分析：在作业详情页提取作业信息并生成评分建议。
- 自动批改全班：按学生列表逐个进入作答页，自动填写分数与评语并保存。
- 单人批改：按姓名筛选并对指定学生执行批改。
- 手动评分标准：支持维护评分项、分值、顺序与建议文本。
- 重新批阅模式：可选择是否包含已批阅作业。
- 规则展开显示：可切换是否展示评分规则细节。
- 一键催交：对未提交作业的学生执行催交通知流程。

### 2.2 附件与识别能力

- 附件图片 OCR：通过 OCR.space 识别图片内容辅助评分。
- 页面文本提取：支持 iframe 内容抓取与作答文本组合分析。

### 2.3 配置与诊断能力

- API Key 安全配置：在插件设置页录入，存储于 `chrome.storage.local`，仅显示掩码。
- 运行日志分级：支持 `debug / info / warn / error / silent`。
- 运行指标诊断：支持查看成功/失败统计与高频失败项。
- 指标清理：支持手动清空运行指标数据。

## 3. 架构与文件职责

- `manifest.json`：扩展入口、权限、注入域名、CSP、图标配置。
- `src/popup/popup.html`：插件弹窗 UI（功能入口、设置与诊断界面）。
- `src/popup/popup.js`：弹窗交互、配置读写、发送消息到 content/background。
- `src/content/content.js`：页面自动化主流程（定位元素、执行批改、提取数据）。
- `src/background/background.js`：服务端调用编排（DeepSeek/OCR）、配置管理、诊断数据汇总。
- `assets/icons/`：扩展图标资源。

## 5. 配置说明

### 5.1 API Key（必需）

1. 打开插件弹窗，进入设置页。
2. 在 API Key 输入框中粘贴 DeepSeek Key 并保存。
3. 保存成功后显示掩码（不回显明文）。

### 5.2 建议使用顺序

1. 先做“AI 作业分析”。
2. 必要时用“手动评分标准”微调规则。
3. 执行“自动批改全班”或“单人批改”。
4. 在诊断页查看运行指标与失败热点。

## 7. 已知边界

- 自动化依赖目标站点页面结构，平台 DOM 变动可能导致流程中断。
- OCR 结果受图片清晰度影响明显。
- 网络波动或接口限流会影响 AI/OCR 调用稳定性。
