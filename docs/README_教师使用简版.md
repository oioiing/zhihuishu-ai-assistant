## 这是什么

这是一个用于智慧树作业批改的浏览器插件，目标是让老师更快完成批改、统计和催交。

## 你可以做什么

- 一键自动批改全班作业
- 批改单个学生作业
- 用 AI 分析作业类型并生成评分标准
- 手动设置评分标准（支持分值、排序、保存）
- 一键催交未提交作业的学生
- 查看班级常见问题与阶段趋势

## 推荐使用流程（最省事）

1. 进入作业详情页。
2. 先点“AI 作业分析”，看系统给出的评分标准。
3. 如有需要，点“手动设置标准”进行微调。
4. 点“开始自动批改”，等待批改完成。
5. 最后查看班级统计，定位共性问题。

## 手动设置评分标准（简要）

在“手动设置评分标准”里可完成：

- 填写作业类型和说明
- 维护评分项（新增、删除、拖拽排序）
- 设置每个评分项分值（0~100，整数）
- 查看总分提示（建议总分 100）
- 填写批改建议与注意事项
- 点击“保存设置”应用到后续自动批改

## 常见提示

- 总分不是 100 也可以保存，但建议调整到 100。
- 图片题依赖 OCR，清晰截图识别效果更好。
- 平台页面结构变化可能影响自动操作，遇到异常可刷新页面重试。

## 出现问题怎么办

- 先刷新页面，再重开插件重试。
- 检查网络是否正常。
- 查看故障文档： [TROUBLESHOOTING.md](TROUBLESHOOTING.md)

## 其他说明

- 插件会调用 AI 接口进行分析与批改。
