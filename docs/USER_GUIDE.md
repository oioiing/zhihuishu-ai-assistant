# 使用指南

## 推荐阅读顺序

1. 先看 [README_教师使用简版.md](README_教师使用简版.md)：快速了解功能范围。
2. 再看 [README.md](../README.md)：查看项目总览与入口说明。
3. 需要深入时看 [功能说明.md](功能说明.md)：了解配置、动作与边界。
4. 遇到异常看 [TROUBLESHOOTING.md](TROUBLESHOOTING.md)：按现象排查。

## 页面与入口

- 作业题目页：右下角浮窗球，单题智能批改
- 学生列表页：左下角红色按钮，自动批改整个班级
- 作业详情页：浮窗菜单中的“AI 分析”按钮

## 单题智能批改

1. 进入任意作业题目页
2. 点击右下角浮窗球
3. 等待 AI 评分与评语生成
4. 手动确认并保存

适用：需要人工审核的作业、开放题、重要作业。

## 自动批改整个班级

1. 进入学生列表页面（看到表格）
2. 点击左下角“启动自动批改”
3. 确认学生数量
4. 等待插件自动进入、填充并保存

适用：标准化试题、班级人数较多的批改场景。

## 作业详情 AI 分析

1. 打开作业详情页
2. 点击浮窗菜单中的“AI 分析”
3. 查看分析结果：作业类型、评分标准、批改建议、常见错误
4. 分析结果会自动用于后续自动批改

提示：分析结果会写入全局状态，自动批改时优先使用这些条件生成评语。

## 重新批阅模式

自动批改菜单中勾选“包括已批阅”，即可将已批作业也纳入自动批改。

注意：会覆盖原有评分与评语，建议在确认评分标准后再启用。

## 常用快捷操作

- 打开控制台：F12
- 停止自动批改：刷新页面或关闭浮窗
