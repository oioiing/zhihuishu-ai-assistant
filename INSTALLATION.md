# 智慧树 AI 助教 - 安装和使用指南

## 📦 安装步骤

### 1. 下载扩展

**方式一：克隆仓库**
```bash
git clone https://github.com/oioiing/zhihuishu-ai-assistant.git
cd zhihuishu-ai-assistant
```

**方式二：下载 ZIP**
- 访问项目页面
- 点击"Code" → "Download ZIP"
- 解压到本地文件夹

### 2. 加载到 Chrome

1. 打开 Chrome 浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 开启右上角的"开发者模式"开关
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹（包含 manifest.json 的文件夹）
6. 确认扩展已加载且处于启用状态

### 3. 配置 API 密钥（重要！）

#### 步骤 A：获取 DeepSeek API Key（必需）

1. **访问 DeepSeek 平台**
   - 网址：https://platform.deepseek.com
   - 注册并登录账户

2. **创建 API 密钥**
   - 进入"API Keys"页面
   - 点击"Create API Key"
   - 复制生成的密钥（格式：`sk-xxxxxxxxxxxxxxxx`）
   - ⚠️ 妥善保管，密钥只显示一次

3. **充值账户**
   - DeepSeek API 按使用量收费
   - 建议先充值少量金额测试（如 ¥10）
   - 查看定价：https://platform.deepseek.com/pricing

#### 步骤 B：获取 OCR.space API Key（可选）

1. **访问 OCR.space**
   - 网址：https://ocr.space/ocrapi
   - 注册免费账户

2. **获取密钥**
   - 免费账户提供 25,000 次/月
   - 在个人中心查看 API Key
   - 复制密钥

#### 步骤 C：在扩展中配置密钥

1. 点击浏览器工具栏中的扩展图标 📚
2. 点击"设置"按钮（⚙️）
3. 在设置页面输入：
   - DeepSeek API Key（必需）
   - OCR.space API Key（可选）
4. （可选）勾选"启用调试模式"以查看详细日志
5. 点击"保存"按钮
6. 等待"设置保存成功"提示

## 🎯 使用指南

### 功能一：截图批改

1. **访问智慧树网站**
   - 打开任意智慧树页面（*.zhihuishu.com 或 *.zhihuishu.cn）
   - 等待页面加载完成

2. **打开悬浮球**
   - 页面右侧会出现 📚 悬浮球
   - 点击悬浮球展开功能列表

3. **选择批改方式**
   - **DOM 自动阅卷**：直接提取网页内容，速度快，准确率高
   - **截图批改**：手动截图，通过 OCR 识别文字内容

4. **查看结果**
   - 系统会自动分析并给出批改建议
   - 包含评分、优点、改进建议等

### 功能二：AI 对话

1. 点击悬浮球的"💬 AI 对话"
2. 在对话框中输入问题
3. AI 助手会提供专业的教学建议

### 功能三：页面摘要

1. 点击悬浮球的"📄 页面摘要"
2. 系统会自动提取页面关键信息
3. 生成结构化的课程分析报告

### 功能四：悬浮球

- 长按悬浮球可拖动位置
- 点击展开/收起功能列表
- 所有功能一键直达

## 🔍 常见问题

### 扩展无法加载？

**症状**：点击"加载已解压的扩展程序"后出现错误

**解决方案**：
1. 确认选择的文件夹包含 `manifest.json`
2. 检查是否开启了"开发者模式"
3. 查看错误提示，根据提示修复

### 悬浮球不显示？

**症状**：访问智慧树网站后看不到悬浮球

**解决方案**：
1. 刷新页面（按 F5）
2. 确认扩展已启用（`chrome://extensions/` 中查看）
3. 确认访问的是智慧树域名（*.zhihuishu.com 或 *.zhihuishu.cn）
4. 打开控制台（F12）查看是否有错误信息

### API 调用失败？

**症状**：提示"API密钥未配置"或"调用失败"

**解决方案**：
1. **检查密钥配置**
   - 打开扩展设置
   - 确认 DeepSeek API Key 已正确填写
   - 密钥格式：`sk-` 开头的长字符串

2. **检查账户余额**
   - 登录 DeepSeek 平台
   - 查看账户余额是否充足
   - 查看 API 调用记录

3. **检查网络连接**
   - 确认可以访问 https://api.deepseek.com
   - 关闭可能影响连接的代理或防火墙

### OCR 识别不准确？

**症状**：截图后识别的文字有误或不完整

**解决方案**：
1. **配置专用 OCR Key**
   - 获取 OCR.space API Key
   - 在设置中配置（不使用默认密钥）

2. **优化截图质量**
   - 确保文字清晰、大小适中
   - 背景与文字有足够对比度
   - 避免截取过多无关内容

3. **使用 DOM 模式**
   - 优先使用"DOM 自动阅卷"功能
   - 直接提取网页内容，准确率 100%

## 🔧 调试技巧

### 开启调试模式

1. 打开扩展设置
2. 勾选"启用调试模式"
3. 保存设置
4. 按 F12 打开控制台
5. 查看详细的运行日志

### 查看错误信息

控制台会显示：
- 🚀 扩展启动信息
- 📡 API 调用状态
- ❌ 错误详情和堆栈
- 🔧 调试模式下的详细日志

### 重置配置

如果遇到问题需要重置：
1. 打开 `chrome://extensions/`
2. 找到"智慧树 AI 助教"
3. 点击"详细信息"
4. 点击"清除存储数据"
5. 重新配置 API 密钥

## 🔒 安全提示

1. **保护 API 密钥**
   - 不要在公共场合展示密钥
   - 不要将密钥提交到 Git 仓库
   - 定期更换密钥

2. **注意账户余额**
   - 定期检查 API 使用量
   - 设置使用额度限制
   - 账户余额不足时及时充值

3. **更新扩展**
   - 定期检查项目更新
   - 拉取最新代码
   - 重新加载扩展
