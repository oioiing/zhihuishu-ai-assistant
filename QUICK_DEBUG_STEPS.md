# 附件无法识别问题 - 快速调试步骤

## 问题描述
显示"✅ 附件处理完成"但AI说"附件下载失败，无法获取题目内容"

## 快速修复步骤（5分钟）

### 第一步：重新加载扩展
1. 打开 `chrome://extensions/`
2. 找到"智慧树AI助手"
3. 点击 **刷新按钮** 🔄
4. 重新加载作业详情页面

### 第二步：查看详细日志
现在扩展已经内置了详细的调试日志。

#### A. 打开浏览器Console
1. 在作业详情页面按 `F12`
2. 切换到 **Console** 标签
3. 点击扩展的"分析作业"按钮

#### B. 查看日志输出
寻找这些关键日志：

**新增：检查提取到的URL（非常重要！）**
```
📎 [URL1] 名称: Ex.7.docx
📎 [URL1] 地址: https://...（完整URL）
📎 [URL1] 方法: 自动点击预览提取
```

**如果DOCX解析成功，应该看到：**
```
📎 [附件1] 文件名: Ex.7.docx
📎 [附件1] 内容长度: 1234 字符
📎 [附件1] 内容预览: Exercise 7 1. Choose...
📎 [作业分析] 附件摘要长度: 400 字符
```

**如果DOCX解析失败，会看到：**
```
📎 [附件1] 内容预览: (无法解析文本 - DOCX文件可能使用了特殊格式...)
⚠️ [附件1] 检测到错误信息: (无法解析文本)
```

### 第三步：查看Service Worker日志（如果需要）
1. 打开 `chrome://extensions/`
2. 找到"智慧树AI助手"
3. 点击 **"Service Worker"** 链接
4. 会打开新的调试窗口
5. 重新点击"分析作业"，观察后台日志

**应该看到：**
```
📄 [Docx解析] 找到 150 个 <w:t> 标签
📄 [Docx解析]最终提取文本 1234 字符
✅ [附件处理] 解析完成: Ex.7.docx (提取 1234 字符)
```

### 第四步：使用调试工具（高级）
1. 在作业详情页面按 `F12` 打开Console
2. 访问 GitHub 仓库，复制 `DEBUG_TOOL.js` 的内容
3. 粘贴到Console并回车
4. 运行命令：
   ```javascript
   quickCheckAttachments()    // 快速检查
   debugAttachments()          // 完整调试
   ```

## 常见问题和解决方案

### 问题1：下载成功但内容为空

**新增：先检查URL类型**
```
📎 [URL1] 地址: https://example.com/preview/12345
⬇️ [附件下载] 开始下载
   ⚠️ 检测到预览页面URL，尝试提取真实文件URL...
   ✅ 预览页面加载成功，长度: 50000 字符
   🎯 找到真实下载URL: https://example.com/download/file.docx
   ✅ 使用真实下载URL替代预览URL
```

**症状1A：URL是预览页面但无法提取真实下载URL**
```
⬇️ [附件下载] 开始下载
   ⚠️ 检测到预览页面URL，尝试提取真实文件URL...
   ⚠️ 未能从预览页面提取下载URL，继续尝试原URL...
   ❌ [附件下载] 失败
   💡 可能的原因：
      1. CORS限制 - 服务器不允许跨域访问
```

**解决方案**：URL是预览页面而非直接下载链接，需要手动下载

**症状1B：DOCX解析成功但内容为空**
```
✅ [附件处理] 下载成功: Ex.7.docx (12345 字节)
📄 [Docx解析] 找到 0 个 <w:t> 标签
📄 [Docx解析] 最终提取文本 0 字符
```

**原因：** DOCX文件格式特殊，当前解析方法无法提取

**临时解决方案：**
1. 点击"手动设置评分标准"按钮
2. 打开附件文件（Ex.7.docx）
3. 复制题目内容
4. 粘贴到"作业内容"字段
5. 点击"生成评分标准"

**长期解决方案：** 等待下一版本更新（将集成专业DOCX解析库mammoth.js）

### 问题2：下载失败
**症状：**
```
❌ [附件处理] 下载失败: Ex.7.docx
```

**原因：** URL提取失败或网络问题

**解决方案：**
1. 尝试在页面上手动点击"预览"按钮，确认附件能正常打开
2. 如果页面预览正常但扩展下载失败，请报告此bug

### 问题3：attachmentSummary为空
**症状：**
```
📎 [作业分析] 附件处理完成
⚠️ [AI分析] attachmentSummary为空，AI将无法看到附件内容！
```

**原因：** 虽然处理了附件，但内容全是错误信息，导致摘要为空

**解决方案：** 查看具体的错误信息（参考问题1或问题2）
