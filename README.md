# zhihuishu-ai-assistant

teacher help: 智能批改作业，帮助老师在指定网页智能批改作业

## Description

This project provides an AI-powered assistant to help teachers automatically grade assignments on the Zhihuishu (智慧树) platform.

## Local Development Setup

### Prerequisites

- Python 3.8 or higher (if Python-based)
- Node.js 14 or higher (if Node.js-based)
- VSCode (recommended)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/oioiing/zhihuishu-ai-assistant.git
   cd zhihuishu-ai-assistant
   ```

2. Install dependencies:
   ```bash
   # For Python projects
   pip install -r requirements.txt
   
   # For Node.js projects
   npm install
   ```

### Running Locally

```bash
# For Python projects
python main.py

# For Node.js projects
npm start
```
