# 智慧树 AI 助教 🧠

基于 DeepSeek AI 的智能教学助手 Chrome 扩展，采用悬浮球交互设计，提供作业自动批改、截图批改、AI对话、页面分析等功能。

## ✨ 核心功能

### ✅ 作业自动批改（v2.0 新功能）
- **三种题型支持**：选择题、填空题、简答题自动识别和批改
- **规则引擎**：基于可自定义规则的本地批改，快速准确
- **AI智能评分**：可选启用DeepSeek AI进行简答题深度分析
- **模糊匹配**：填空题支持同义词、相似度匹配，更人性化
- **批量批改**：一键批改所有题目，提高效率
- **详细反馈**：每题提供得分、反馈和改进建议
- **可视化结果**：直观的UI显示批改结果和统计数据
- **导出报告**：支持导出详细批改报告
- **灵活配置**：支持自定义批改规则和评分标准

### 🎯 智能识别作业
- **自动检测**页面中的作业区域，无需手动选择
- 支持多种识别方式：CSS选择器、关键词匹配
- 智能过滤无关区域（导航栏、页脚等）
- 置信度评分系统，自动排序最可能的作业区域
- 一键选择识别到的区域进行批改

### ⚙️ 自定义批改规则
- **批改重点**：可选择语法、拼写、标点、词汇、句式、逻辑、格式、内容
- **批改严格程度**：宽松（鼓励为主）、适中（平衡指导）、严格（细致批改）
- **自定义指令**：输入特定的批改要求，AI根据指令个性化批改
- 设置持久化保存，下次使用自动应用

### 📸 截图批改
- 截取当前屏幕并使用AI分析作业内容
- OCR识别 → AI批改分析
- 支持中英文混合识别，智能排版
- 多语言自动检测（简体中文、英文、繁体中文）

### 💬 AI对话
- 与DeepSeek AI直接对话
- 实时聊天，支持教学问答
- 输入问题，回车或点击发送

### 📊 页面分析
- 分析当前页面内容和关键词
- 提取页面摘要，识别教学要点
- 快速了解页面主要内容

### 🔍 知识图谱
- 构建页面知识关系图
- 提取概念关系，显示权重
- 可视化知识结构

## 🚀 安装使用

### 前置要求

- Chrome浏览器（版本88+）
- 支持Manifest V3的浏览器
- Node.js 和 npm（开发者模式）

### 开发者安装步骤

1. **克隆仓库**
   ```bash
   git clone https://github.com/oioiing/zhihuishu-ai-assistant.git
   cd zhihuishu-ai-assistant
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **构建项目**
   ```bash
   npm run build
   ```

4. **加载到Chrome**
   - 打开Chrome浏览器扩展管理页面 (`chrome://extensions/`)
   - 开启右上角的"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择 `zhihuishu-ai-assistant` 文件夹

### 使用方法

#### 基础使用
1. **查看悬浮球** - 安装后访问任意网页，右侧会出现蓝色📚悬浮球
2. **点击悬浮球** - 打开功能面板，包含7个核心功能
3. **拖拽移动** - 长按悬浮球可拖拽到任意位置
4. **选择功能** - 点击对应功能按钮开始使用

### 新功能使用指南

#### ✅ 作业自动批改（v2.0 重点新功能）

**适用页面**：
- https://hike-teaching-center.polymas.com/homeworkCorrect/*
- 任何包含作业题目的页面

**使用步骤**：

1. **打开作业批改页面**
   - 访问支持的作业页面
   - 确保页面完全加载

2. **启动批改系统**
   - 点击悬浮球
   - 选择 "✅ 作业批改"
   - 系统自动识别页面中的题目

3. **查看识别结果**
   - 系统显示识别到的题目数量
   - 显示总分和题型分布
   - 可选择开始批改或显示批改面板

4. **开始批改**
   - 点击 "🚀 开始批改所有题目"
   - 系统自动批改所有题目
   - 实时显示批改进度

5. **查看结果**
   - 批改完成后显示总分和统计
   - 每题下方显示详细反馈
   - 支持导出批改报告

**题型支持**：

1. **选择题**
   - 自动比对标准答案
   - 精确匹配评分
   - 配置：大小写敏感、空格处理

2. **填空题**
   - 模糊匹配支持
   - 同义词识别
   - 相似度评分（可配置阈值）
   - 多空题按比例给分

3. **简答题**
   - 规则引擎：字数、关键词评分
   - AI服务（可选）：智能深度分析
   - 提供详细反馈和建议

**高级配置**：

```javascript
// 自定义批改规则示例
const customRules = {
    fillInBlank: {
        synonyms: {
            '中国': ['中华人民共和国', 'China'],
            '北京': ['首都', 'Beijing']
        },
        similarityThreshold: 0.85  // 相似度阈值
    },
    shortAnswer: {
        minWordCount: 50,
        useAI: true  // 启用AI评分
    }
};
```

详细配置请参考：[自定义批改规则指南](./自定义批改规则指南.md)

#### 🎯 智能识别作业
1. 打开包含作业的网页
2. 点击悬浮球 → 选择"🎯 智能识别作业"
3. 系统自动识别页面中的作业区域
4. 从识别结果中选择要批改的区域
5. 查看AI批改结果

#### ⚙️ 自定义批改规则
1. 在智能识别页面点击"⚙️ 批改设置"
2. 选择批改重点（可多选）：语法、拼写、标点、词汇等
3. 选择批改严格程度：宽松、适中、严格
4. 输入自定义批改指令（可选）
5. 点击"💾 保存设置"
6. 设置将在下次批改时自动生效

#### 使用场景示例
- **英语作文批改**：重点选择"语法+词汇+句式"，严格程度"适中"
- **数学作业批改**：重点选择"逻辑+格式"，严格程度"严格"
- **鼓励式批改**：严格程度选择"宽松"，自定义指令"多鼓励学生"

## ⚙️ API配置

### 获取API密钥
1. 访问 https://platform.deepseek.com
2. 注册账户并完成验证
3. 创建API密钥（格式：sk-xxxxxxxx...）
4. ⚠️ **重要** - 为账户充值余额（API调用需要付费）

### 配置插件
1. 打开 `background_fixed.js` 文件
2. 找到以下代码：
   ```javascript
   const API_KEY = "sk-6f2c1a0e4f6c4274a3abd1754777655b";
   ```
3. 替换为您的真实API密钥：
   ```javascript
   const API_KEY = "您的真实API密钥";
   ```
4. 保存文件并重新加载扩展

## 🔧 故障排除

### 常见问题

**Q: 作业批改功能不可用？**
- 确保页面是作业批改页面（如 `hike-teaching-center.polymas.com/homeworkCorrect/*`）
- 检查浏览器控制台是否有错误（按F12查看）
- 尝试刷新页面重新加载扩展
- 确认已运行 `npm run build` 编译TypeScript代码
- 检查 `homework-grading-bundle.js` 文件是否存在

**Q: 题目识别不准确？**
- 刷新页面确保完全加载
- 检查页面HTML结构是否符合常见模式
- 查看控制台日志了解识别详情
- 可以手动使用"截图批改"功能作为备选

**Q: 填空题判分过于严格？**
- 调整相似度阈值（默认0.8，可降低到0.7）
- 添加同义词映射
- 参考 [自定义批改规则指南](./自定义批改规则指南.md)

**Q: AI评分不工作？**
- 检查是否设置了 `useAI: true`
- 确认DeepSeek API密钥已配置
- 检查账户余额是否充足
- 查看网络连接是否正常
- 参考 [AI服务接入说明](./AI服务接入说明.md)

**Q: 编译TypeScript失败？**
- 确保Node.js版本 >= 14
- 删除 `node_modules` 和 `package-lock.json`
- 重新运行 `npm install`
- 运行 `npm run build` 查看详细错误

**Q: 智能识别找不到作业区域？**
- 确保页面完全加载
- 检查页面是否包含作业相关内容
- 使用"📸 手动截图"功能手动选择区域

**Q: 批改规则设置后不生效？**
- 确保点击了"💾 保存设置"按钮
- 设置会在下次批改时自动生效
- 可以点击"🔄 恢复默认"重置设置

**Q: 悬浮球不显示？**
- 刷新页面重新加载
- 检查扩展是否正确安装并启用
- 查看浏览器控制台错误信息

**Q: API调用失败？**
- 检查API密钥是否正确配置
- 确认账户余额充足
- 检查网络连接状态

**Q: 截图功能无响应？**
- 确认页面完全加载
- 检查是否在特殊页面（chrome://等）
- 查看控制台错误信息

### 错误代码
| 错误类型 | 原因 | 解决方案 |
|---------|------|---------|
| 401 认证失败 | API密钥无效 | 检查密钥格式，重新配置 |
| 403 访问拒绝 | 余额不足 | 充值DeepSeek账户 |
| 429 频率限制 | 调用过频 | 等待重试或实施限流 |
| 网络连接失败 | 网络问题 | 检查网络连接 |
| TypeScript编译错误 | 代码错误 | 查看编译输出，修正错误 |
