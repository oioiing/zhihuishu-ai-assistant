# 智慧树 AI 助教 🧠

基于 DeepSeek AI 的智能教学助手 Chrome 扩展，专为智慧树（Zhihuishu）平台设计。采用现代化悬浮球交互，提供 DOM/截图双模式批改、AI 对话、页面分析及知识图谱构建等核心功能。

## ✨ 核心功能

### 🤖 DOM 批改（全新推荐功能）
- **直接提取**：直接从网页 DOM 结构中提取题目和学生回答文本，识别率 100%。
- **精准度高**：规避 OCR 误识别问题，完美支持复杂排版。
- **作答检测**：智能提取输入框、文本域、单/多选及富文本编辑器的学生内容。
- **报告生成**：自动生成包含原始文本和 AI 批改建议的详细报告，支持下载保存。

### 🎯 智能识别作业
- **自动检测**：一键识别页面中的作业区域，智能过滤导航、页脚、广告等噪音。
- **置信度评分**：基于 CSS 选择器和语义关键词进行双重评分，精准定位。
- **双模选择**：针对每个识别到的区域，可自由选择“DOM 批改”或“截图批改”。

### ⚙️ 自定义批改规则
- **多维重点**：自定义关注语法、拼写、标点、词汇、句式、逻辑、格式或内容。
- **严格程度**：提供宽松（鼓励为主）、适中（平衡指导）、严格（细致挑错）三种风格。
- **个性化指令**：支持输入特殊教学要求（如：“重点关注虚拟语气的使用”）。

### 📸 截图批改
- **区域捕捉**：支持手动拖拽选择或智能区域截取。
- **增强 OCR**：集成中英繁多语言 OCR 识别，具备智能文本自动排版功能。

### 💬 AI 对话 & 页面分析
- **实时答疑**：随时唤起 AI 助教进行教学问答。
- **内容摘要**：智能提取页面核心关键词和教学要点报告。
- **知识图谱**：自动构建当前页面的知识概念图谱，可视化教学逻辑。

## 🚀 安装使用

### 安装步骤
1. 下载并解压本项目。
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`。
3. 开启“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择项目文件夹。

### 使用流程
1. **唤起**：访问智慧树页面，点击右侧出现的🧠蓝色悬浮球。
2. **选择模式**：点击“🎯 智能识别作业”。
3. **执行批改**：在识别出的区域下方，点击“✨ DOM 批改”开始分析。
4. **调整设置**：点击“⚙️ 批改设置”定制 AI 的评价标准。

## ⚙️ API 配置

1. 访问 [DeepSeek 开放平台](https://platform.deepseek.com) 获取 API Key。
2. 打开 `background.js`。
3. 将 `API_KEY` 变量替换为您的真实密钥：
   ```javascript
   const API_KEY = "YOUR_REAL_API_KEY_HERE";
   ```
4. 在扩展管理页面点击“重新加载”。

## 🛠️ 技术细节

### 动画与交互设计
- **Stagger Effect**：功能列表项采用 50ms 延迟依次滑入，提供流畅的视觉体验。
- **毛玻璃效果**：UI 界面采用 `backdrop-filter: blur(12px)` 实现高级感官效果。
- **状态环**：球体周围设有动态彩色旋转环，实时反馈 AI 思考状态。

### 文件结构
- `manifest.json`: 插件定义（Manifest V3）。
- `content.js`: UI 渲染、DOM 提取与页面交互核心逻辑。
- `background.js`: 后台通信、DeepSeek API 调用与 OCR 集成。
- `popup.html`: 插件管理入口。

## 🔒 安全与合规
- **CSP 兼容**：所有交互均使用 `addEventListener` 绑定，严格遵守内容安全策略。
- **数据隐私**：提取的作业内容仅用于 API 分析，不进行本地持久化以外的存储。
