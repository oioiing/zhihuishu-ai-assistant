# 知慧树智能作业阅卷助手 v2

这是一个用于知慧树教学平台的 Chrome 扩展，提供 AI 作业分析、单题批改和自动批改能力，帮助教师提升批改效率。

## 主文档入口

本项目文档已收敛，优先阅读：

- `QUICK_START.md`：唯一主文档（安装、使用、排障、优化说明）

## 3 分钟安装

1. 下载或克隆本仓库到本地。
2. 打开 Chrome，进入 `chrome://extensions/`。
3. 开启右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择本项目根目录。
5. 打开 `QUICK_START.md` 按步骤验证与使用。

## 功能概览

- 作业详情 AI 分析
- 单题智能批改
- 班级自动批改
- 重试、缓存、日志等稳定性增强能力
