# 智慧树 AI 助教 🧠

基于 DeepSeek AI 的智能教学助手 Chrome 扩展，采用悬浮球交互设计，提供截图批改、AI对话、页面分析等功能。专为智慧树在线教育平台设计。

## ✨ 核心功能

### 🎯 智能识别作业 (核心功能)
- **自动检测**页面中的作业区域，无需手动选择。
- **双模式批改**：
    - **DOM 快速阅卷** (推荐)：直接提取网页 DOM 文字内容进行分析，准确度高，速度快，不受图片质量影响。
    - **截图批改**：截取作业区域并通过 OCR 识别文字，适用于包含复杂公式或特殊布局的作业。
- 支持多种识别方式：CSS 选择器、关键词匹配。
- 智能过滤无关区域（导航栏、页脚等）。
- 置信度评分系统，自动推荐最可能的作业区域。

### ⚙️ 自定义批改规则
- **批改重点**：可选择语法、拼写、标点、词汇、句式、逻辑、格式、内容。
- **批改严格程度**：宽松（鼓励为主）、适中（平衡指导）、严格（细致批改）。
- **自定义指令**：输入特定的批改要求，AI 根据指令个性化批改。
- 设置持久化保存，下次使用自动应用。

### 💬 AI 对话
- 与 DeepSeek AI 直接对话，支持教学问答、学习指导。
- 实时聊天界面，支持回车发送。

### 📊 页面分析
- 自动提取当前页面的核心关键词和内容摘要。
- 帮助教师快速了解课程要点。

### 🔍 知识图谱
- 构建页面知识关系图，提取概念关系并可视化知识结构。

### 📄 页面摘要
- 生成结构化的课程分析报告，包含课程档案、核心内容、知识图谱和学习攻略。

## 🚀 安装使用

### 安装步骤
1. 下载本项目文件夹。
2. 打开 Chrome 浏览器扩展管理页面 (`chrome://extensions/`)。
3. 开启右上角的"开发者模式"。
4. 点击"加载已解压的扩展程序"。
5. 选择包含 `manifest.json` 的项目文件夹。

### 配置 API 密钥
1. 访问 [DeepSeek 开放平台](https://platform.deepseek.com) 注册并获取 API Key。
2. 在 `background.js` 中找到以下位置：
   ```javascript
   const API_KEY = "YOUR_API_KEY_HERE";
   ```
3. 将 `YOUR_API_KEY_HERE` 替换为您真实的 API Key。
4. 保存文件并重新加载扩展程序。

### 使用方法
1. **查看悬浮球** - 安装后访问智慧树网页，右侧会出现 📚 悬浮球。
2. **展开菜单** - 点击悬浮球，展示功能列表。
3. **开始批改** - 选择 "🎯 智能识别作业"，系统将自动标记页面中的作业区。
4. **选择模式** - 在识别出的区域下方，点击 "✨ DOM 快速阅卷" 或 "📸 截图批改"。

## 🎨 交互设计

- **悬浮球**: 极简圆形设计，支持拖拽，具备毛玻璃质感。
- **功能列表**: Stagger 展开动画，圆润视觉元素。
- **面板系统**: 现代化卡片式设计，现代化交互体验。

## 🛠️ 技术实现

- **前端**: Chrome Extension Manifest V3, DOM API, CSS3 Animations.
- **后端**: DeepSeek API (deepseek-chat), OCR.space API (作为截图识别备选).
- **识别算法**: 结合 CSS 选择器匹配与关键词加权评分。

## 💡 最佳实践

- **英语作文批改**: 重点选择 "语法+词汇+句式"，程度设为 "适中"。
- **数学作业批改**: 重点选择 "逻辑+格式"，程度设为 "严格"，并配合自定义指令。
- **低年级批改**: 程度设为 "宽松"，并在自定义指令中加入 "多鼓励学生"。

## 🔧 常见问题

- **Q: 找不到作业区域？**
  - 请确保页面已加载完成，或尝试刷新页面。如仍无法识别，可使用 "📸 截图批改" 手动选择。
- **Q: API 调用失败？**
  - 请检查 API Key 是否配置正确，并确认账户余额充足。
