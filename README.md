# 智慧树 AI 助教 🧠

基于 DeepSeek AI 的智能教学助手 Chrome 扩展，采用悬浮球交互设计，提供 DOM 自动阅卷、智能作业识别、截图批改、AI 对话、页面分析等功能。

## ✨ 核心功能

### 🔍 DOM 自动阅卷（新功能）
- **直接提取**：从网页 DOM 中直接提取文本内容，避免 OCR 识别误差。
- **高准确度**：原始文本直接发送至 AI，批改更精准。
- **智能排版**：保留网页原始文本结构，解析更清晰。

### 🎯 智能识别作业
- **自动检测**：页面中的作业区域，无需手动选择。
- **多种方式**：支持 CSS 选择器、关键词匹配。
- **置信度评分**：自动排序最可能的作业区域，智能过滤无关区域（导航栏、页脚等）。

### ⚙️ 自定义批改规则
- **批改重点**：可选择语法、拼写、标点、词汇、句式、逻辑、格式、内容。
- **严格程度**：宽松（鼓励为主）、适中（平衡指导）、严格（细致批改）。
- **自定义指令**：输入特定的批改要求，AI 根据指令个性化批改。

### 📸 截图批改
- **区域截图**：支持手动选择区域或对自动识别区域进行截图。
- **OCR 识别**：使用 OCR.space API 进行多语言文字识别。
- **智能分析**：AI 分析图片中的作业内容并提供批改建议。

### 💬 AI 对话
- **实时答疑**：与 DeepSeek AI 直接对话，解决教学问题。
- **上下文理解**：支持持续对话，提供针对性的学习指导。

### 📊 页面分析与摘要
- **内容提取**：智能分析当前页面核心关键词和摘要。
- **知识图谱**：构建页面知识关系图，可视化知识结构。
- **结构化报告**：生成包含课程档案、核心内容、学习攻略的结构化分析。

## 🚀 安装使用

### 安装步骤
1. 下载本项目文件夹。
2. 打开 Chrome 浏览器扩展管理页面 (`chrome://extensions/`)。
3. 开启右上角的 "开发者模式"。
4. 点击 "加载已解压的扩展程序"，选择 `zhihuishu-ai-assistant` 文件夹。

### 使用方法
1. **查看悬浮球** - 安装后访问网页，右侧会出现📚悬浮球。
2. **选择功能** - 点击悬浮球展开菜单，选择 "🎯 智能识别作业" 或其他功能。
3. **批改作业** - 在识别出的作业区域中，点击 "🔍 DOM 自动批改" 或 "📸 截图批改"。
4. **设置规则** - 点击 "⚙️ 批改设置" 自定义批改标准。

## ⚙️ API 配置

1. 访问 [DeepSeek 开放平台](https://platform.deepseek.com) 获取 API Key。
2. 打开 `background.js` 文件，配置 `API_KEY`：
   ```javascript
   const API_KEY = "您的 API Key";
   ```
3. 重新加载扩展程序。

## 🛠️ 技术架构

### 核心技术
- **Manifest V3**: 最新 Chrome 扩展规范。
- **DeepSeek Chat API**: 提供核心 AI 逻辑。
- **OCR.space API**: 提供图片文字识别。
- **Content Scripts**: 负责页面 DOM 操作、悬浮 UI 注入。

### 交互设计
- **悬浮球**: 采用毛玻璃质感（Backdrop-filter），支持拖拽和优雅的展开动画。
- **功能面板**: 现代化卡片式设计，支持即时反馈和结果保存。

## 📁 文件结构
- `manifest.json`: 扩展配置文件。
- `content.js`: 页面交互、DOM 提取、悬浮 UI 逻辑。
- `background.js`: 后台 Service Worker，负责 API 通信和任务调度。
- `popup.html`: 插件弹窗界面。

## 🔧 故障排除
- **悬浮球不显示**：请刷新页面或检查插件是否已启用。
- **识别不准**：尝试使用 "📸 手动截图" 功能或调整批改设置。
- **API 错误**：请检查 API Key 是否正确及账户余额是否充足。
