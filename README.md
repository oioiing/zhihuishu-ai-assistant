# 智慧树 AI 助教 🧠

基于 DeepSeek AI 的智能教学助手 Chrome 扩展，采用悬浮球交互设计，提供截图批改、AI对话、页面分析等功能。

## ✨ 核心功能

### 🎯 智能识别作业（新功能）
- **自动检测**页面中的作业区域，无需手动选择
- 支持多种识别方式：CSS选择器、关键词匹配
- 智能过滤无关区域（导航栏、页脚等）
- 置信度评分系统，自动排序最可能的作业区域
- 一键选择识别到的区域进行批改

### ⚙️ 自定义批改规则（新功能）
- **批改重点**：可选择语法、拼写、标点、词汇、句式、逻辑、格式、内容
- **批改严格程度**：宽松（鼓励为主）、适中（平衡指导）、严格（细致批改）
- **自定义指令**：输入特定的批改要求，AI根据指令个性化批改
- 设置持久化保存，下次使用自动应用

### 📸 截图批改
- 截取当前屏幕并使用AI分析作业内容
- OCR识别 → AI批改分析
- 支持中英文混合识别，智能排版
- 多语言自动检测（简体中文、英文、繁体中文）

### 💬 AI对话
- 与DeepSeek AI直接对话
- 实时聊天，支持教学问答
- 输入问题，回车或点击发送

### 📊 页面分析
- 分析当前页面内容和关键词
- 提取页面摘要，识别教学要点
- 快速了解页面主要内容

### 🔍 知识图谱
- 构建页面知识关系图
- 提取概念关系，显示权重
- 可视化知识结构

## 🚀 安装使用

### 安装步骤
1. 打开Chrome浏览器扩展管理页面 (`chrome://extensions/`)
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `zhihuishu-ai-assistant` 文件夹

### 使用方法
1. **查看悬浮球** - 安装后访问任意网页，右侧会出现蓝色🧠悬浮球
2. **点击悬浮球** - 打开功能面板，包含6个核心功能
3. **拖拽移动** - 长按悬浮球可拖拽到任意位置
4. **选择功能** - 点击对应功能按钮开始使用

### 新功能使用指南

#### 🎯 智能识别作业
1. 打开包含作业的网页
2. 点击悬浮球 → 选择"🎯 智能识别作业"
3. 系统自动识别页面中的作业区域
4. 从识别结果中选择要批改的区域
5. 查看AI批改结果

#### ⚙️ 自定义批改规则
1. 在智能识别页面点击"⚙️ 批改设置"
2. 选择批改重点（可多选）：语法、拼写、标点、词汇等
3. 选择批改严格程度：宽松、适中、严格
4. 输入自定义批改指令（可选）
5. 点击"💾 保存设置"
6. 设置将在下次批改时自动生效

#### 使用场景示例
- **英语作文批改**：重点选择"语法+词汇+句式"，严格程度"适中"
- **数学作业批改**：重点选择"逻辑+格式"，严格程度"严格"
- **鼓励式批改**：严格程度选择"宽松"，自定义指令"多鼓励学生"

## ⚙️ API配置

### 获取API密钥
1. 访问 https://platform.deepseek.com
2. 注册账户并完成验证
3. 创建API密钥（格式：sk-xxxxxxxx...）
4. ⚠️ **重要** - 为账户充值余额（API调用需要付费）

### 配置插件
1. 打开 `background_fixed.js` 文件
2. 找到以下代码：
   ```javascript
   const API_KEY = "sk-6f2c1a0e4f6c4274a3abd1754777655b";
   ```
3. 替换为您的真实API密钥：
   ```javascript
   const API_KEY = "您的真实API密钥";
   ```
4. 保存文件并重新加载扩展

## 🔧 故障排除

### 常见问题

**Q: 智能识别找不到作业区域？**
- 确保页面完全加载
- 检查页面是否包含作业相关内容
- 使用"📸 手动截图"功能手动选择区域

**Q: 批改规则设置后不生效？**
- 确保点击了"💾 保存设置"按钮
- 设置会在下次批改时自动生效
- 可以点击"🔄 恢复默认"重置设置

**Q: 悬浮球不显示？**
- 刷新页面重新加载
- 检查扩展是否正确安装并启用
- 查看浏览器控制台错误信息

**Q: API调用失败？**
- 检查API密钥是否正确配置
- 确认账户余额充足
- 检查网络连接状态

**Q: 截图功能无响应？**
- 确认页面完全加载
- 检查是否在特殊页面（chrome://等）
- 查看控制台错误信息

### 错误代码
| 错误类型 | 原因 | 解决方案 |
|---------|------|---------|
| 401 认证失败 | API密钥无效 | 检查密钥格式 |
| 403 访问拒绝 | 余额不足 | 充值账户 |
| 429 频率限制 | 调用过频 | 等待重试 |
| 网络连接失败 | 网络问题 | 检查网络 |
