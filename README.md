# 智慧树 AI 助教 🧠

基于 DeepSeek AI 的智能教学助手 Chrome 扩展，采用悬浮球交互设计，提供智能阅卷、AI对话、页面分析等核心功能。

## ✨ 核心功能

### 🎯 智能作业阅卷
- **DOM 自动阅卷（新）**：直接提取网页文本和学生输入内容，无需截图，识别率 100%，速度极快。
- **智能识别区域**：自动检测页面中的作业区域，支持 CSS 选择器和关键词匹配。
- **截图批改**：支持手动/自动截图，集成 OCR.space 高精度文字识别。
- **多语言支持**：完美支持中英文混合、繁体中文识别与批改。

### ⚙️ 自定义批改规则
- **多维度设置**：可选择语法、拼写、标点、词汇、句式、逻辑、格式、内容作为批改重点。
- **严格程度**：提供宽松（鼓励为主）、适中（平衡指导）、严格（细致批改）三种模式。
- **自定义指令**：支持输入个性化指令，如“重点关注时态，忽略拼写错误”。

### 💬 AI 对话与辅助
- **AI 助教对话**：与 DeepSeek AI 实时交流，解答教学难题。
- **页面智能分析**：提取核心关键词，识别教学要点。
- **结构化摘要**：生成 Markdown 格式的课程分析报告，包含课程档案、核心内容、知识图谱及学习攻略。

## 🚀 快速开始

### 安装步骤
1. **下载扩展**
   - 克隆或下载本项目代码到本地
   
2. **加载到浏览器**
   - 打开 Chrome 浏览器扩展管理页面 (`chrome://extensions/`)
   - 开启右上角的"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择项目所在的文件夹

3. **配置 API 密钥**（重要）
   - 点击浏览器工具栏中的插件图标 📚
   - 点击"设置"按钮（⚙️）
   - 在设置页面配置以下密钥：

### API 密钥获取

#### DeepSeek API Key（必需）
1. 访问 [DeepSeek 平台](https://platform.deepseek.com)
2. 注册并登录账户
3. 进入 API Keys 页面创建新密钥
4. 复制密钥（格式：`sk-xxxxxxxxxxxxxxxx`）
5. 在插件设置中粘贴并保存

#### OCR.space API Key（可选，用于图片识别）
1. 访问 [OCR.space](https://ocr.space/ocrapi)
2. 注册免费账户（每月 25,000 次免费调用）
3. 获取 API Key
4. 在插件设置中粘贴并保存

> **注意**：如不配置 OCR Key，系统将使用默认密钥，但可能有使用限制。

## 📖 使用指南

1. **悬浮球交互**：访问任意网页后，右侧会出现 📚 悬浮球。点击展开功能列表，长按可自由拖拽。
2. **DOM 自动阅卷**：点击“智能识别作业”，在识别到的区域下选择“🔍 DOM 自动阅卷”，系统将直接读取题目和您的答案进行深度分析。
3. **截图批改**：选择“📸 截图批改”手动框选区域，AI 将通过 OCR 识别内容并给出修改建议。

## 🛠️ 技术架构

- **核心技术**：Manifest V3, DeepSeek Chat API, OCR.space API.
- **模块设计**：
  - `content.js`: 负责 DOM 提取、UI 注入及交互逻辑。
  - `background.js`: 负责后台服务、跨域 API 调用及复杂分析流。
  - `popup.html`: 插件快捷操作界面和设置配置。
  - `manifest.json`: 扩展配置文件，定义权限和内容脚本。

### 安全性说明

- ✅ **API 密钥加密存储**：所有 API 密钥通过 `chrome.storage.sync` 安全存储
- ✅ **不提交敏感信息**：`.gitignore` 确保本地配置文件不会被提交
- ✅ **权限最小化**：扩展仅请求必要的浏览器权限
- 🔒 **请勿分享密钥**：请妥善保管您的 API 密钥，不要与他人分享

## 🔧 常见问题

### 安装和配置问题

**Q: 扩展无法加载？**
- ✅ 确认已开启"开发者模式"
- ✅ 确认选择的文件夹包含 `manifest.json`
- ✅ 查看扩展管理页面是否有错误提示

**Q: 悬浮球不显示？**
- ✅ 刷新智慧树网页（按 F5）
- ✅ 确保扩展已在 `chrome://extensions/` 中启用
- ✅ 打开浏览器控制台（F12）查看是否有错误信息
- ✅ 确认访问的是智慧树相关域名（*.zhihuishu.com）

**Q: API 密钥配置后不生效？**
- ✅ 检查密钥格式是否正确（DeepSeek 密钥以 `sk-` 开头）
- ✅ 点击保存按钮后等待"设置保存成功"提示
- ✅ 刷新网页后重试

### 功能问题

**Q: 识别失败？**
- ✅ 确保页面内容已完全加载
- ✅ 若 DOM 模式无法识别，请尝试截图模式
- ✅ 检查网络连接是否正常
- ✅ 确认 API 密钥已正确配置

**Q: API 调用报错？**
- ✅ 检查 DeepSeek API Key 是否正确配置
- ✅ 确认 DeepSeek 账户余额充足
- ✅ 查看控制台是否有详细错误信息
- ✅ 尝试开启调试模式查看详细日志

**Q: OCR 识别不准确？**
- ✅ 确保图片中文字清晰可见
- ✅ 尝试截取更小的区域，专注于文字部分
- ✅ 配置 OCR.space API Key 以获得更好的服务
- ✅ 手动输入作业内容作为替代方案

### 调试技巧

**开启调试模式**：
1. 点击插件图标打开弹窗
2. 点击"设置"按钮
3. 勾选"启用调试模式"
4. 保存设置
5. 打开浏览器控制台（F12）查看详细日志

**查看错误日志**：
1. 按 F12 打开浏览器开发者工具
2. 切换到 Console 标签页
3. 查看错误信息和堆栈跟踪
4. 根据错误提示进行排查
