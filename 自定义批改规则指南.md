# 自定义批改规则指南

## 概述

智慧树作业自动批改系统提供了灵活的规则引擎，支持自定义批改规则，满足不同学科和教学场景的需求。

## 批改规则类型

### 1. 选择题规则 (Multiple Choice)

选择题采用精确匹配策略，可配置以下选项：

#### 可配置参数

- **caseSensitive** (boolean): 是否区分大小写
  - `true`: 严格匹配大小写 (A ≠ a)
  - `false`: 忽略大小写 (A = a) [默认]

- **trimWhitespace** (boolean): 是否去除空格
  - `true`: 自动去除前后空格 [默认]
  - `false`: 保留空格

#### 示例配置

```javascript
multipleChoice: {
    caseSensitive: false,  // 不区分大小写
    trimWhitespace: true   // 去除空格
}
```

#### 评分标准

- 答案完全匹配 → 满分
- 答案不匹配 → 0分

### 2. 填空题规则 (Fill in Blank)

填空题支持模糊匹配，提供更智能的判分策略。

#### 可配置参数

- **fuzzyMatch** (boolean): 是否启用模糊匹配
  - `true`: 允许相似答案通过 [默认]
  - `false`: 只接受完全匹配

- **ignoreCase** (boolean): 是否忽略大小写
  - `true`: 忽略大小写 [默认]
  - `false`: 严格区分大小写

- **trimWhitespace** (boolean): 是否去除空格
  - `true`: 自动去除前后空格 [默认]
  - `false`: 保留空格

- **synonyms** (object): 同义词映射
  - 定义可接受的同义词替换
  - 格式：`{ "标准答案": ["同义词1", "同义词2"] }`

- **similarityThreshold** (number): 相似度阈值 (0-1)
  - 范围：0.0 - 1.0
  - 默认：0.8 (80% 相似度)
  - 建议值：
    - 严格：0.9
    - 适中：0.8
    - 宽松：0.7

#### 示例配置

```javascript
fillInBlank: {
    fuzzyMatch: true,
    ignoreCase: true,
    trimWhitespace: true,
    synonyms: {
        '是': ['对', 'yes', '正确', '√'],
        '否': ['错', 'no', '不对', '×'],
        '中国': ['中华人民共和国', 'China'],
        '北京': ['首都', 'Beijing']
    },
    similarityThreshold: 0.8
}
```

#### 评分策略

- 完全匹配 → 满分
- 同义词匹配 → 满分
- 相似度 ≥ 阈值 → 满分
- 部分填空正确 → 按比例给分
- 全部错误 → 0分

#### 相似度计算

系统使用 Levenshtein 距离算法计算字符串相似度：

```
相似度 = 1 - (编辑距离 / 最大长度)
```

### 3. 简答题规则 (Short Answer)

简答题支持基于规则的初步评分和AI服务评分。

#### 可配置参数

- **keywordMatching** (boolean): 是否启用关键词匹配
  - `true`: 检查答案中的关键词 [默认]
  - `false`: 不检查关键词

- **keywords** (array): 关键词列表
  - 手动指定的关键词数组
  - 如为空，自动从标准答案提取

- **minWordCount** (number): 最少字数
  - 默认：10
  - 少于此字数会扣分

- **maxWordCount** (number): 最多字数
  - 默认：1000
  - 超过此字数会扣分

- **useAI** (boolean): 是否使用AI评分
  - `true`: 调用AI服务进行智能评分
  - `false`: 仅使用规则引擎 [默认]

#### 示例配置

```javascript
shortAnswer: {
    keywordMatching: true,
    keywords: ['主要', '重要', '特点', '意义'],  // 可选，自动提取
    minWordCount: 50,
    maxWordCount: 500,
    useAI: false  // 设为 true 启用AI评分
}
```

#### 规则引擎评分标准

1. **基础分** (50%)
   - 满足最低字数要求

2. **关键词加分** (每个关键词 10%)
   - 最多加 50%

3. **字数评估**
   - 少于最少字数：扣 20%
   - 超过最多字数：扣 10%
   - 字数合适：不扣分

4. **及格线**
   - 60% 及以上为通过

#### AI评分模式

当 `useAI: true` 时，系统将调用DeepSeek AI进行智能评分：

- 评分范围：0-10分
- 提供详细反馈
- 给出改进建议
- 评估置信度

## 完整配置示例

### 示例1：严格批改模式

适用于期末考试、重要测验等场景。

```javascript
const strictRules = {
    multipleChoice: {
        caseSensitive: true,    // 严格区分大小写
        trimWhitespace: true
    },
    fillInBlank: {
        fuzzyMatch: false,      // 不启用模糊匹配
        ignoreCase: false,      // 区分大小写
        trimWhitespace: true,
        synonyms: {},           // 不接受同义词
        similarityThreshold: 1.0  // 必须完全匹配
    },
    shortAnswer: {
        keywordMatching: true,
        keywords: [],
        minWordCount: 100,      // 要求更多字数
        maxWordCount: 500,
        useAI: true             // 启用AI精准评分
    }
};
```

### 示例2：宽松批改模式

适用于日常练习、自测等场景。

```javascript
const lenientRules = {
    multipleChoice: {
        caseSensitive: false,
        trimWhitespace: true
    },
    fillInBlank: {
        fuzzyMatch: true,
        ignoreCase: true,
        trimWhitespace: true,
        synonyms: {
            '是': ['对', 'yes', '正确', '对的', '是的'],
            '否': ['错', 'no', '不对', '错误', '不是']
        },
        similarityThreshold: 0.7  // 降低相似度要求
    },
    shortAnswer: {
        keywordMatching: true,
        keywords: [],
        minWordCount: 20,       // 字数要求较低
        maxWordCount: 1000,
        useAI: false            // 使用规则引擎快速评分
    }
};
```

## 代码集成

### 在代码中使用自定义规则

```javascript
// 导入批改引擎
import { GradingEngine } from './gradingEngine';

// 创建自定义规则
const customRules = {
    fillInBlank: {
        synonyms: {
            '中国': ['中华人民共和国', 'China', 'PRC']
        },
        similarityThreshold: 0.85
    }
};

// 初始化批改引擎
const engine = new GradingEngine(customRules);

// 或者更新现有规则
engine.updateRules(customRules);
```

## 最佳实践

### 1. 根据科目调整规则

- **数学/理科**：严格模式，不启用模糊匹配
- **语文/文科**：适中模式，允许同义词
- **英语**：区分大小写，启用AI评分
- **编程**：完全匹配，区分大小写

### 2. 根据年级调整规则

- **低年级**：宽松模式，鼓励为主
- **中年级**：适中模式，平衡指导
- **高年级**：严格模式，细致批改

### 3. 根据作业类型调整

- **日常练习**：宽松模式，快速反馈
- **周测验**：适中模式
- **期中/期末**：严格模式，精准评分

### 4. AI评分的使用建议

- **优先使用场景**：简答题、作文、开放性问题
- **谨慎使用场景**：客观题、计算题
- **注意事项**：
  - 确保API密钥已配置
  - 评分时间会增加
  - 需要网络连接
  - 成本考虑

## 故障排除

### Q: 填空题判分过于严格？
A: 降低 `similarityThreshold` 或添加同义词

### Q: 简答题给分太低？
A: 降低 `minWordCount` 或调整关键词列表

### Q: AI评分不工作？
A: 检查 `useAI` 设置和API密钥配置

### Q: 如何恢复默认设置？
A: 不传递任何参数创建 `GradingEngine` 实例

## 扩展开发

### 添加自定义评分逻辑

```javascript
// 继承 GradingEngine 类
class CustomGradingEngine extends GradingEngine {
    // 重写批改方法
    async gradeShortAnswer(question) {
        // 自定义逻辑
        const result = await super.gradeShortAnswer(question);
        
        // 添加额外判断
        if (question.studentAnswer.includes('创新')) {
            result.score += 1; // 额外加分
            result.suggestions.push('答案体现了创新思维');
        }
        
        return result;
    }
}
```
