### 1. 智能识别目标作业区域

**实现位置：** `content.js` - `handleAutoDetectHomework()` 函数

**核心功能：**
- 自动检测页面中的作业区域
- 支持多种识别方式：
  - CSS选择器匹配（`.homework-box`, `.question-box` 等）
  - 关键词匹配（"作业"、"练习"、"答题"等）
  - 智能过滤无关区域（导航栏、页脚等）

### 2. 对目标区域内的文字进行识别

**实现位置：** `background.js` - `performOCR()` 函数

**核心功能：**
- 使用OCR.space API进行文字识别
- 支持多语言识别（简体中文、英文、繁体中文）
- 智能文本排版和格式化

**OCR流程：**
```javascript
async function performOCR(imageData) {
    // 1. 尝试多种语言识别
    const languages = ['chs', 'eng', 'cht'];
    
    for (let lang of languages) {
        try {
            // 2. 调用OCR API
            const ocrResult = await callOnlineOCRWithLanguage(imageData, lang);
            
            if (ocrResult && ocrResult.trim().length > 0) {
                // 3. 自动排版
                const formattedText = formatOCRText(ocrResult, lang);
                return formattedText;
            }
        } catch (error) {
            // 继续尝试下一种语言
        }
    }
}
```

**文本排版优化：**
- 中文文本：处理标点符号、中英文混合空格
- 英文文本：合并分割的单词、修复句子间空格
- 通用清理：去除多余空格和换行

### 3. 用免费的AI模型进行批改

**实现位置：** `background.js` - `analyzeTextContent()` 函数

**使用的AI模型：**
- **DeepSeek Chat API**（免费额度）
- 模型：`deepseek-chat`
- 特点：支持中文、性价比高、响应快

**批改流程：**
```javascript
async function analyzeTextContent(text) {
    // 1. 获取用户自定义的批改规则
    const correctionRules = await getCorrectionRules();
    
    // 2. 构建AI提示词
    const prompt = `
    批改要求：${strictnessPrompt}
    重点关注：${correctionRules.focusAreas.join('、')}
    特殊要求：${correctionRules.customInstructions}
    
    作业内容：${text}
    `;
    
    // 3. 调用DeepSeek API
    const response = await callDeepSeekAPI(prompt);
    
    return response;
}
```

**批改输出格式：**
- 📝 作业类型识别
- 📊 评分建议（满分100分）
- ✅ 优点分析
- ⚠️ 需要改进的地方
- 💡 具体修改建议
- 📚 学习建议
